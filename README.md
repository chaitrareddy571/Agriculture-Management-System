🌾 Agriculture Management System

📌 Overview

The Agriculture Management System is a web-based application designed to streamline agricultural activities for farmers, buyers, and administrators. It enables farmers to list crops, buyers to browse and place orders, and administrators to manage and monitor transactions efficiently.

---

🚀 Setup Steps

1. Clone the repository:
   
   git clone https://github.com/yourusername/agriculture-management-system

2. Navigate to the project folder:
   
   cd agriculture-management-system

3. Open the project in your IDE (Eclipse / IntelliJ)

4. Configure database:
   
   - Install SQL Server
   - Create database and import provided SQL file

5. Run the application:
   
   - Execute main Java files / deploy on server

---

💻 Tech Stack Used

- Frontend: Java Applets / GUI (AWT, Swing)
- Backend: Java
- Database: SQL Server
- Technologies: JDBC, Embedded SQL Queries
- Tools: Eclipse IDE

---

✨ Features Implemented

- 👨‍🌾 Farmer Module:
  
  - Add and manage crops
  - View orders

- 🛒 Buyer Module:
  
  - Browse available crops
  - Place orders

- 🛠️ Admin Module:
  
  - Monitor transactions
  - Manage users and system data

- 🔐 Basic Authentication System

- 📊 Database integration for real-time data
